# bosskung-skills

A minimal Claude plugin marketplace with generated sample skills.

## Marketplace

- Open `/tmp/workspace/Thadchet/bosskung-skills/marketplace/index.html` in a browser
- Or serve the folder:

```bash
cd /tmp/workspace/Thadchet/bosskung-skills/marketplace
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Included generated skills

- Prompt Polisher
- Meeting Summarizer
- Bug Hunter
