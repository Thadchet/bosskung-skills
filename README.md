# bosskung-skills

A minimal Claude plugin marketplace with generated sample skills.

## Marketplace

- Open `marketplace/index.html` in a browser
- Or serve the folder:

```bash
cd marketplace
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Included generated skills (examples)

- Prompt Polisher
- Meeting Summarizer
- Bug Hunter
